## Installation
Install with [jitpack](https://jitpack.io/#com.vperi/kotlin-coroutines-lib/):

### Gradle

```gradle
repositories {
  maven { url 'https://jitpack.io' }
}

dependencies {
  compile 'com.vperi:kotlin-coroutines-lib:<latest-version-here>'
}

```
